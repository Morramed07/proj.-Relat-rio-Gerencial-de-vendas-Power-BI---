# Projeto Relatório Gerencial de Vendas com Power Bi
Este projeto foi proposto pelo bootcamp da suzano - análise de dados com Power BI, onde aborda os conceitos e métodos de análise de dados com o POWER BI. Nesse projeto damos nossos primeiros passos para fazermos relatório simplificados e de fácil visualização com diferentes visões dos dados disponibilizados da SAMPLE FINANCIALS.
